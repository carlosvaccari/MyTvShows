## My TV Shows

* In this project was used MVVM arch

### 3rd party libraries
* Glide
* Retrofit
* Koin
* Mockk
* Robolectric
* Espresso

### How to run
* UnitTests -> ./gradlew test
* Application -> ./gradlew installDebug

### Work in progress, but currently has the following features:
* Shows list
* Shows search
* Show details
* Episode details
* Favorites
